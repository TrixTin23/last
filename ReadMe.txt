1. Buat Virtual env "python -m venv appenv"
2. Masuk ke env "cd appenv"
3. Copy Folder App ke appenv
4. aktifkan appenv melalui cmd "scripts\activate"
5. Install Flask "pip install Flask"
6. Install Pandas "pip install Pandas"
7. Install request "pip install request"
8. masuk ke dalam App dengan cara cd App
9. Run app.py dengan cara klik run atau ketiik "python app.py" pada terminal
